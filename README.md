# Dog-Breed-Classifier-5-Class-Deep-Learning-Model
A deep learning model that accurately classifies 5 popular dog breeds: French Bulldog, German Shepherd, Golden Retriever, Poodle, and Yorkshire Terrier using Convolutional Neural Networks.
# 🐶 5 Dog Breed Classifier

A **Deep Learning** web application that predicts one of **5 popular dog breeds** from an uploaded image using **TensorFlow & Keras**.

## 📋 Supported Breeds

- **French Bulldog**
- **German Shepherd**
- **Golden Retriever**
- **Poodle**
- **Yorkshire Terrier**

---

## ✨ Features

- **Real-time Dog Breed Prediction** using uploaded images
- Beautiful and user-friendly **Streamlit Web Interface**
- Built with **TensorFlow + Keras**
- High accuracy CNN model
- Easy to deploy (Streamlit Cloud / Hugging Face / Render)
- Confidence score displayed for each prediction

---

## 🛠️ Technologies Used

- **Python 3.9+**
- **TensorFlow & Keras** (for model training & inference)
- **NumPy** (for image preprocessing)
- **Streamlit** (for web app)
- **Pillow** (for image handling)
